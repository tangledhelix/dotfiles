External comands

  :r!<cmd>        read in the result of a command

  !cmd            pipe selection through command, replace with result

  :echo glob($VIMRUNTIME . "/syntax/*.vim")      Show all known syntaxes

  command | vim -     edit command's output in vim as a buffer
